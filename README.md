# Personal Introduction Program

A beginner-friendly Python project that collects user information and displays a personalized welcome message. This project was created to practice the fundamentals of Python programming, including variables, user input, strings, and formatted output.

---

## 📌 Project Overview

The **Personal Introduction Program** asks users for basic information such as their name, age, city, and hobby. It then displays a friendly personalized introduction using Python variables and f-strings.

This project is part of **Week 1 – Python Basics**.

---

## 🎯 Objectives

* Learn Python syntax
* Understand variables and data types
* Practice using `input()` and `print()`
* Use f-strings for formatted output
* Build a simple interactive Python application

---

## 🛠️ Technologies Used

* Python 3.x
